# TinyGo 项目

这是一个基于 TinyGo 的项目，旨在为微控制器和WebAssembly平台提供高性能的Go语言开发环境。

## 项目特点

- 轻量级的Go编译器
- 支持微控制器平台
- WebAssembly支持
- 低内存占用

## 快速开始

### 安装要求

- Go 1.19+
- TinyGo编译器

### 编译示例

```bash
tinygo build -o main.wasm -target wasm main.go
```
